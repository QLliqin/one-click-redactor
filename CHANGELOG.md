# 更新记录

## Android v1.0.0

- 新增可直接安装的原生 Android 应用，支持 Android 8.0 及以上系统。
- 移植桌面版全部内置脱敏类型、角色姓名联动和补充规则。
- 支持 DOC/DOCX/DOCM、XLSX/XLSM、PPTX/PPTM、TXT/CSV 离线处理。
- 新增系统文件/文件夹选择、任务卡片、进度、日志、结果打开和报告生成。
- Android 应用不申请网络权限，原文件保持不变。
- 新增 JVM 单元测试和 Android Runtime 真机环境测试。

## v1.1.1 专业版

- 新增专属“盾牌 + 脱敏文档”应用图标。
- 新增统一输出目录，并保留原目录输出模式。
- 默认跳过已脱敏文件和报告，避免重复处理。
- 新增多选移除、清空日志、打开结果和完成后自动打开目录。
- 新增批次总报告开关和命令行 `--output`、`--include-redacted` 参数。
- 同名文件输出自动避让，不再发生覆盖。
- 修复部分 Word/WPS 生成的 `.doc` 写出后无法重新打开的问题。

## v1.0.0 轻量版

- 支持 DOC、DOCX、DOCM、XLSX、XLSM、PPTX、PPTM、TXT、CSV。
- 支持个人身份、联系方式、地址、企业信息和法院/公安文书角色姓名脱敏。
- 保留原文件，在原目录生成脱敏副本和报告。
